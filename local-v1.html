Local-only v1 · Android + Windows focus

Browse, preview, and organize local files without cloud sign-in.

Use user-granted local access to build a session library for videos, images, PDFs, and HTML files. Metadata stays local in IndexedDB, and v1 avoids direct filesystem write-back promises.

Choose the workflow your browser supports. Folder access and saved handle reuse are capability-gated.

What v1 does

Ingests local content, renders it by type, and keeps stack/tags/notes/ratings/favorite metadata locally.

What v1 defers

No direct file moves, arbitrary deletes, or sync/write-back semantics. Removing items only affects this app view.

Platform footnotes

iOS

Local file access is not the target path for this version. Expect repeated selection prompts and reduced capability support.

macOS

This version prioritizes Android and Windows flows. Some persistent or folder-based workflows may need repeated authorization or may not be available.

Privacy

No cloud-provider authentication is used here. Everything in v1 remains local to this browser profile.
